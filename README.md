# 🚀 Blog API - FastAPI

A simple Blog REST API built using **FastAPI** and **PostgreSQL**.

## ✨ Features

- PostgreSQL Integration
- CRUD Operations
- JWT Authentication
- Protected Routes
- Pagination
- Search Functionality
- GitHub Version Control

## 🛠️ Tech Stack

- FastAPI
- PostgreSQL
- SQLAlchemy
- Pydantic
- JWT
- Uvicorn
- Python

## 🚀 Installation

```bash
git clone https://github.com/your-username/Blog-API-FastAPI.git
cd Blog-API-FastAPI
```

Create a virtual environment:

```bash
python -m venv venv
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the server:

```bash
uvicorn main:app --reload
```

Open:

- Swagger UI: `http://127.0.0.1:8000/docs`
- ReDoc: `http://127.0.0.1:8000/redoc`
